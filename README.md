# Organization Hierarchy Frontend

A modern web application for visualizing and managing organization employee hierarchy with unlimited depth levels, built with Next.js, Redux Toolkit, and Mantine UI.

## 🚀 Technologies

- Next.js 14
- Redux Toolkit
- Mantine UI
- TailwindCSS
- React Hook Form
- Axios

## 📋 Features

- ✅ Tree visualization with unlimited depth
- ✅ Create, edit, delete positions  
- ✅ CEO badge and report counts
- ✅ Responsive design
- ✅ Toast notifications

## 📦 Installation

```bash
git clone https://github.com/Rediet-Geremew/pis-employee-hierarchy.git
cd pis-employee-hierarchy
npm install
npm run dev
